# 接口文档

服务默认地址：

```text
http://localhost:8000
```

## 健康检查

```http
GET /api/health
```

返回：

```json
{"status": "ok"}
```

## 上传教材

```http
POST /api/textbooks/upload
Content-Type: multipart/form-data
```

字段：

- `files`: 一个或多个教材文件，支持 PDF、Markdown、TXT、DOCX。

返回：

```json
{
  "files": [
    {
      "textbook_id": "book_xxx",
      "filename": "生理学.pdf",
      "title": "生理学",
      "format": "pdf",
      "size": 1024,
      "total_pages": 520,
      "total_chars": 385000,
      "chapter_count": 12,
      "status": "completed",
      "error": null
    }
  ]
}
```

## 教材列表

```http
GET /api/textbooks
```

返回教材摘要数组，不包含章节全文。

## 教材详情

```http
GET /api/textbooks/{textbook_id}
GET /api/textbooks/{textbook_id}?include_content=true
```

默认返回章节结构；`include_content=true` 时返回章节正文。

## 导出统一 JSON

```http
GET /api/textbooks/{textbook_id}/export
```

返回赛题要求的完整结构，包含全部章节正文。

## 删除教材

```http
DELETE /api/textbooks/{textbook_id}
```

删除上传文件、解析缓存、知识图谱和向量索引。

## 生成知识图谱

```http
POST /api/textbooks/{textbook_id}/knowledge-graph
```

请求体：

```json
{
  "chapter_ids": ["ch_001"],
  "max_chapters": 20,
  "force": false
}
```

返回：

```json
{
  "textbook_id": "book_xxx",
  "title": "生理学",
  "model": "Qwen/Qwen3-30B-A3B-Instruct-2507",
  "generated_at": "2026-05-10T00:00:00Z",
  "chapter_count": 20,
  "nodes": [],
  "edges": [],
  "extraction_errors": []
}
```

## 异步知识图谱任务

```http
POST /api/textbooks/{textbook_id}/knowledge-graph/jobs
GET /api/knowledge-graph/jobs/{job_id}
```

用于长教材抽取进度展示。

## 聚合知识图谱

```http
GET /api/knowledge-graph
```

读取已生成的单本图谱并进行跨教材合并。

## RAG 索引

```http
POST /api/rag/index/{textbook_id}
POST /api/rag/index-all
GET /api/rag/status
```

用于建立和查看教材向量索引。

## RAG 问答

```http
POST /api/rag/query
```

请求体：

```json
{
  "question": "细胞膜的主要功能是什么？",
  "textbook_ids": ["book_xxx"],
  "top_k": 5
}
```

返回：

```json
{
  "question": "细胞膜的主要功能是什么？",
  "answer": "答案正文，带 [1] 来源编号。",
  "sources": [
    {
      "source_index": 1,
      "textbook_id": "book_xxx",
      "textbook_title": "生理学",
      "chapter_id": "ch_001",
      "chapter_title": "第一章 绪论",
      "page": 12,
      "excerpt": "来源片段...",
      "score": 0.82
    }
  ],
  "model": "Qwen/Qwen3-30B-A3B-Instruct-2507"
}
```
